# Queues VK MiniApp!
BackEnd + FrontEnd of queues VkMiniApp :)

  <img width="300" height="300" src="https://user-images.githubusercontent.com/48258107/132247308-7b258fde-8396-4f0b-9e0d-9939fff66356.png">
</p>

<br/>
<h3> Подробнее о возможностях:</h1>
1. Перемешивание очереди.
<br/>
2. Получение уведомлений о продвижении в очереди от нашего бота.
<br/>
3. Добавление администраторов очереди.
<br/>
4. Отправка сообщения всем участникам очереди.
<br/>
5. Удаление участника очереди.
<br/>
6. Приглашение в очередь по спец коду или же по прямой ссылке на очередь.
<br/>
7. И многое другое!
<br/>
